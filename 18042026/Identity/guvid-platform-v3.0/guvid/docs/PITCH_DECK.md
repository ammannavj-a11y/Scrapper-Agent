# GUVID v3.0 — Investor & Partner Pitch

## The Problem (3 slides worth)

**Every organisation re-verifies every person, every time.**

- A graduate applies to 20 companies → 20 separate KYC verifications
- Each verification costs ₹200–₹2,000 (Signzy/Karza/HyperVerge pricing)
- User shares raw national ID (Aadhaar, SSN) with each verifier — privacy nightmare
- No standard: university credentials, EPFO data, PAN — all siloed
- **Market cost: $8.3B/year in India alone in redundant identity verification**

---

## The Solution

**GUVID: Verify once. Present everywhere. Cryptographically.**

```
Citizen verifies with government APIs once
        ↓
GUVID token issued + anchored to Hyperledger Fabric
        ↓
Employer scans GUVID → cryptographic proof in <500ms
        ↓
No re-verification. No raw data sharing. No API calls.
```
